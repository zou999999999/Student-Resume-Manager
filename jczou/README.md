## 📄 本模块核心文件清单
- `main.cpp` — 系统主程序入口
- `mainwindow.h/cpp` — 控制器与自动数据保存集成绑定逻辑
- `mainwindow.ui` — 前端界面描述文件
- `BDT220new.pro` — 集成项目构建文件
- `test_runner.cpp` — 单元测试程序
- `BDT220_独立运行包` (目录) 以及 `BDT220_独立运行包.zip` (压缩包) — 包含自检启动脚本 `双击我检测并运行.bat` 的独立运行资产

## 🚀 编译与运行方式
本文件夹包含完整的 Qt 项目结构，确保该模块可以被独立编译及测试。
1. 使用 Qt Creator 打开本目录下的 `BDT220new.pro` 项目文件。
2. 配置好编译器（如 Desktop Qt 6.x.x MinGW 64-bit）。
3. 点击「构建项目」并运行。
