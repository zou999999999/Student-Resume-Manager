# 手动测试检查清单

## 功能一：学业数据管理

- [ ] 打开程序，默认加载上次保存的数据（如无则空白）
- [ ] 输入课程名称、学分、成绩，点击「添加课程」，表格中正确显示
- [ ] 课程名称为空时点击添加，应弹出提示
- [ ] 学分为 0 时添加，应能正常添加
- [ ] 成绩为 0 时添加，应能正常添加
- [ ] 成绩为 100 时添加，应能正常添加
- [ ] 连续添加 10 门课程，表格行数正确
- [ ] 选中一行点击「删除选中课程」，该行消失
- [ ] 未选中任何行点击删除，应弹出提示
- [ ] 点击「计算 GPA」，显示正确的加权 GPA（手动验算）
- [ ] 删除所有课程后计算 GPA，显示 "--" 或 0
- [ ] 点击「保存数据」，状态栏显示成功
- [ ] 关闭程序重新打开，数据应自动恢复（验证持久化）

## 功能二：简历素材库

- [ ] 选择类型「实习经历」，填写标题/机构/日期/描述，点击「添加经历」，表格中正确显示
- [ ] 分别测试四种类型：实习经历、项目经历、校园经历、其他
- [ ] 标题为空时点击添加，应弹出提示
- [ ] 起止日期颠倒（结束日期早于开始日期）—— 应允许（不做校验，用户自行负责）
- [ ] 描述为空时添加 —— 应能正常添加
- [ ] 日期留空（不清除默认值直接添加）
- [ ] 连续添加 20 条经历，表格行数正确
- [ ] 选中一行点击「删除选中经历」，该行消失
- [ ] 切换到学业管理标签页再切回，数据保持
- [ ] 切换到导出标签页，摘要数据正确

## 功能三：简历导出

- [ ] 在导出标签页填写姓名、联系方式、求职意向、技能
- [ ] 点击「一键导出 PDF」，弹出文件保存对话框
- [ ] 选择保存路径，成功生成 PDF 文件
- [ ] 用 PDF 阅读器打开，检查排版和内容完整性
- [ ] 姓名为空时点击导出，应弹出提示
- [ ] 无任何课程时导出，GPA 显示 -- 或 0
- [ ] 无任何经历时导出，经历板块不显示
- [ ] 课程包含特殊字符（如 `<script>`）—— 应被 HTML 转义，不会执行
- [ ] 经历描述包含换行符 —— PDF 中应保留换行
- [ ] 取消文件保存对话框 —— 不应崩溃

## 边界与异常测试

- [ ] 程序安装在中文路径下，能正常启动
- [ ] 程序安装在带空格的路径下，能正常启动
- [ ] JSON 文件被手动损坏后，程序启动不崩溃（数据加载失败时静默使用空数据）
- [ ] 导出路径磁盘无剩余空间 —— 导出失败，显示错误提示
- [ ] 导出路径为只读目录 —— 导出失败，显示错误提示
- [ ] 窗口缩放、最大化、还原 —— 布局自适应
- [ ] 快速连续点击添加按钮 —— 不应重复添加
